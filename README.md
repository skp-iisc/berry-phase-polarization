# Berry Phase Polarization

1. Ivo Souza, Jorge Iniguez, and David Vanderbilt; PRL 89, 117602 (2002)
2. Ivo Souza, Jorge Iniguez, and David Vanderbilt; PRB 69, 085106 (2004)
