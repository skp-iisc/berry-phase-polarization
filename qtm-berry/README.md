# Calculations on Quantum MASALA (QTM)

To run these, you need to use the `ksham.py` and `scf.py` files in this folder in your `src/qtm/dft/` folder inside `QuantumMASALA` root directory.

The SCF calculation (`scf_qtm.py`) matches with the QE SCF (`scf_qe.in`). This can be seen compaing the .out files in `outputs/` folder. The same folder contains the calculations on AlAs (`alas.py`) and AlP (`alp.py`).

Benchmarking: QE outputs (in `../qe-berry/` folder), *Souza, Iniguez, and Vanderbilt; PRL 89, 117602 (2002)*.
